# Portfolio Gabriel Tavares

## Sobre o Projeto

Portfolio pessoal desenvolvido para apresentar minha trajetória profissional como Desenvolvedor Full-Stack Júnior. O projeto foi construído com HTML, CSS e JavaScript puro, seguindo princípios de design moderno com foco em experiência do usuário e performance.

## Tecnologias Utilizadas

- **HTML5** - Estruturação semântica do conteúdo
- **CSS3** - Estilização com variáveis CSS, animações e design responsivo
- **JavaScript Vanilla** - Interatividade e funcionalidades dinâmicas
- **Google Fonts** - Space Grotesk, Inter e JetBrains Mono
- **Font Awesome** - Ícones para redes sociais e elementos visuais

## Funcionalidades

- **Design Responsivo** - Adaptação automática para diferentes tamanhos de tela
- **Tema Claro/Escuro** - Alternância entre temas com persistência visual
- **Scroll Progress** - Barra de progresso indicando posição na página
- **Typewriter Effect** - Animação de digitação no título principal
- **Terminal Interativo** - Apresentação dinâmica de informações
- **Animações de Scroll** - Elementos revelados suavemente ao rolar a página
- **Navegação Suave** - Links internos com scroll suave e destaque de seção ativa
- **Menu Mobile** - Navegação adaptada para dispositivos móveis

## Seções do Portfolio

- **Hero** - Apresentação principal com terminal interativo
- **Sobre** - Minha história, formação e estatísticas
- **Experiências** - Linha do tempo profissional com detalhes das empresas
- **Projetos** - Cards com trabalhos desenvolvidos
- **Serviços** - Áreas de atuação e especialidades
- **Conhecimentos** - Tecnologias, ferramentas e certificações

## Design System

O projeto utiliza um sistema de design consistente baseado em:

- **Design Tokens** - Variáveis CSS para cores, espaçamento e tipografia
- **Grid System** - Estrutura de grid responsiva
- **Componentes Reutilizáveis** - Cards, botões e elementos modulares
- **Animações** - Transições suaves com cubic-bezier personalizado

## Responsividade

O portfolio é totalmente responsivo, adaptando-se a:

- **Desktop** - Layout completo com grid e timeline
- **Tablet** - Ajustes de grid e espaçamento
- **Mobile** - Menu hambúrguer e layout em coluna única

## Como Executar Localmente

1. Clone o repositório:
```bash
git clone https://github.com/tavareesdev/portfolio.git
```

2. Navegue até o diretório do projeto:
```bash
cd portfolio
```

3. Abra o arquivo `index.html` no seu navegador

## Estrutura do Projeto

```
portfolio/
├───docs                # Documentos
│   └───readmes         # Documentação dos projetos
├───images              # Imagens dos projetos
└───index.html          # Página principal

```

## Otimizações Implementadas

- Lazy loading para imagens
- Meta tags para SEO
- Design acessível (aria-label, focus-visible)
- Performance otimizada com CSS minimalista
- Transições suaves com GPU acceleration
